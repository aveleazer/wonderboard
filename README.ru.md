# Wonderboard

Виртуальный совет директоров на базе Claude. 7 ИИ-советников, которые спорят с вами и друг с другом.

> *«Единственный инструмент, который скажет то, что вы не хотите слышать.»*

## Что это

У вас есть бизнес-вопрос. Вы созываете Совет. Шляпник проводит интервью, чтобы понять контекст. Затем 7 Мудрецов — каждый со своей оптикой — независимо анализируют ситуацию, спорят друг с другом и дают конкретные планы действий. Шляпник подводит итог и добавляет одну перпендикулярную мысль, переворачивающую ситуацию.

Никаких поддакиваний. Никакого поиска консенсуса. Настоящее несогласие от архетипов, которые думают по-разному.

## Быстрый старт

```bash
git clone https://github.com/aveleazer/wonderboard.git
cd wonderboard
ANTHROPIC_API_KEY=sk-... node server.js
```

Открывается в браузере на `localhost:3737`. Всё.

При первом запуске создаётся `context.md` — опишите там свой бизнес. Этот контекст включается в промпт каждого советника.

## Требования

- Node.js 18+
- Переменная окружения `ANTHROPIC_API_KEY`

## Как проходит сессия

1. **Ваш вопрос** — о чём хотите получить совет
2. **Интервью Шляпника** — уточняющие вопросы для извлечения контекста
3. **Раунд 1** — каждый Мудрец отвечает независимо + задаёт свой уточняющий вопрос
4. **Дополнительный опросник** — Шляпник компилирует вопросы Мудрецов в фокусированную анкету
5. **Раунд 2** — Мудрецы читают ответы друг друга из Раунда 1 + ваши ответы, выдают финальные позиции с планами действий
6. **Синтез** — Шляпник подводит итог: консенсус, разногласия и одна безумная перпендикулярная мысль

## Настройка

- `context.md` — ваш бизнес-контекст (создаётся автоматически, в git не попадает)
- `profiles/*.md` — персоны советников (можно редактировать)
- `prompts/*.md` — шаблоны промптов
- 10 языков, определяются автоматически из браузера

## Архитектура

Ноль зависимостей. Только встроенные модули Node.js.

| Файл | Что делает |
|------|-------------|
| `server.js` | HTTP-сервер, вызовы Claude API, логика сессий |
| `ui.html` | Одностраничный интерфейс — без фреймворков, без сборки |
| `profiles/` | Системные промпты персон |
| `prompts/` | Шаблоны промптов |

Философия проекта — в [PRINCIPLES.md](PRINCIPLES.md).

## Тестовый режим

Введите **test question** в поле вопроса, чтобы запустить полную сессию с мок-данными. Без вызовов API, без затрат токенов. Удобно для знакомства с интерфейсом.
